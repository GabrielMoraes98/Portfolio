Olá, eu sou Gabriel Moraes, um desenvolvedor frontend apaixonado por tecnologia e soluções criativas. Estou em constante formação e aprendizado, pois gosto muito da área de programação. Projetos em destaque

Aplicação Web para Pesquisa e Visualização de Usuários do GitHub

Neste projeto utiliza-se a API do GitHub para pesquisar usuários, repositórios e eventos a partir do nome de usuário digitado.

Gerador de Conselhos

O projeto consiste em um gerador de conselhos que, ao clicar no botão, é executado a função que faz a requisição da API e retorna um conselho para o usuário.

Cards Pokemón

Este projeto foi desenvolvido utilizando HTML, CSS e JavaScript puro, inspirado no anime Pokémon, afim de apresentação no estilo de cards. A aplicação apresenta um card de cada vez, contendo a imagem de um Pokémon e uma breve descrição sobre ele.

X-MEN

O projeto, desenvolvido com HTML, CSS e JavaScript puro, tem como tema os X-Men. O objetivo é criar uma interface interativa onde os usuários podem explorar e selecionar personagens da série X-Men.
